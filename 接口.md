## 投票接口

+   来源检查
+   获取IP

POST 方法，数据格式：
```json
{
    // 数字对应数据表中的寝室 ID
    "chosenItem": [ 1, 2, 3, 4 /*, ... */ ]
}
```

返回：
```json
/* 成功 */
{
    "msg": "success"
}

/* 失败 */
{
    "msg": "failed"
}
```
